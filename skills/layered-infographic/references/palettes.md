# Scientific-infographic palettes (主色 / 辅色 / 强调色)

Restrained academic systems for 科研信息图 / 教学图 / 综述图. Avoid cheap blue-purple
"tech" gradients and over-saturation. Pick ONE system per image; define it as CSS
tokens in the SVG `<style>` block so it can be swapped in one place.

## Nature 系 · 克制学术
- 主色: 灰蓝 `#3a4a5a` · 雾蓝 `#6b8299` · 冷灰 `#aebfce` · 象牙白 `#f4f2ec`
- 辅色: 柔雾粉 `#e6cfc9` · 浅鼠尾草绿 `#bcccb8` · 淡紫灰 `#cabfd0`
- 强调: 珊瑚红 `#e07a5f` · 深蓝 `#1d3557` · 金黄 `#e9c46a`（小面积）
- 气质: 克制、干净、理性、学术

## Science 系 · 理性机制
- 主色: 蓝灰 `#5b6e80` · 中性蓝 `#3f6f9f` · 青灰 `#7d8f92`
- 辅色: 浅紫 `#c8c2dd` · 豆沙粉 `#d8b8b2` · 淡青 `#bcd6d4`
- 强调: 砖红 `#b5502f` · 芥黄 `#d9a531` · 深青蓝 `#1f5f6b`
- 气质: 清楚、功能导向、机制感强

## 柔和粉蓝 · 综述友好
- 主色: 雾霾蓝 `#8aa0b8` · 浅蓝灰 `#c2cedd`
- 辅色: 灰粉 `#dcc4c8` · 浅紫 `#cbc3de` · 鼠尾草绿 `#bccab6`
- 强调: 莓红 `#c0455f` · 钴蓝 `#2f5fb0` · 杏黄 `#e2b06a`
- 气质: 温和、高级、综述友好

## 外科 / 医学说明图 · 浅色
